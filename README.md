# Arovia landing page

One-page bespoke landing page prepared for Arovia, Inc. (Houston, TX), booth SS4 at eMerge Americas Miami 2026. The page carries a 60/30/10 orchestration read scoped to a two-to-four-week advisory engagement, plus a booked Calendly for Matt Creamer.

Built in Arovia's own visual language: Assistant typeface, warm yellow (#FAB542) accents on near-black (#07090B) text, white paper background, teal (#0D6E5F) supporting callouts. No Eduba branding at the top. Eduba identified only in the footer line.

Intended URL path: /for/arovia

## Files

- `index.html`: single-page layout, semantic HTML, no JS
- `styles.css`: Arovia-native CSS tokens, mobile-responsive from 375px up
- `logo.svg`: Arovia wordmark, copied from Brand Pack/assets

## Notes

- All external URLs are real `<a>` elements with `target="_blank" rel="noopener"`.
- No pricing, retainer, or revenue-share numbers anywhere on the page.
- NLP Logix mention carries the mandated credibility line: "in machine learning since 2011" and "over 150 data scientists."
- CTA points to Matt Creamer's 30-minute Calendly: https://calendly.com/thecro-eduba/30min
